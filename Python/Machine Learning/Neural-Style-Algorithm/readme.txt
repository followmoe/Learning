This program uses the Limited-memory-Broyden-Fletcher-Goldfarb-Shanno-Algorithm (L-BFGS-Algorithm) to transfer the style of an image A (style image) to an image B (content image), while maintaining the original content of image B.

Usage:
Content and style image needed. Both images are reshaped to 400x400 for processing, result will be same resolution as content image.

1. Content image is named "portrait.jpg", style image is named "style.jpg"
2. Place both in folder "img"
3. Run script

Source:
Chollet, Francois. Deep Learning with Python and Keras. mitp 2018. ISBN 978-3-95845-838-3
